# 📱 Mobile Sales Dashboard — Power BI

![Dashboard Preview](preview/Dashboard.png)

## 📌 Overview
An interactive **Mobile Sales Dashboard** built in **Microsoft Power BI**, 
analyzing mobile phone sales data across Indian cities.

Developed as part of the **30 Days Power BI Micro Course** by 
[Sathish Dhawale](https://skillcourse.in), Founder of SkillCourse.

---

## 📊 Key Metrics
| Metric | Value |
|---|---|
| Total Sales | ₹769M |
| Total Quantity | 19K |
| Total Transactions | 4K |
| Average Price | ₹40K |

---

## 🔍 Dashboard Features
- **Total Sales by City** — Map visualization across Indian cities
- **Total Quantity by Day** — Day-wise trend line chart
- **Total Sales by Mobile Model** — Top 5 models bar chart
- **Total Transactions by Payment Method** — UPI, Cash, Debit & Credit Card pie chart
- **Customer Ratings** — Distribution across 1–5 star ratings
- **Total Sales by Day Name** — Weekday-wise funnel chart
- **Brand-wise Summary** — Apple, OnePlus, Samsung, Vivo, Xiaomi

---

## 🔽 Filters Available
- Mobile Model
- Payment Method
- Brand
- City
- Month (January – December)
